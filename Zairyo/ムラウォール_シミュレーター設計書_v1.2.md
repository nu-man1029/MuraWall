## 6. 計算ロジック（完全版）

### 共通ルール
- 数量はすべて `ceil()` で切り上げ
- カラーは主材・フィニッシュそれぞれ1:1で連動
- クリヤーは16kg缶優先、端数を2kg缶で補完

### 6-1. 内装（MURA WALL）計算式

```javascript
const area = 施工面積（㎡）

// 工程1: 下地処理（石膏ボード）
const pate  = Math.ceil(area / 62.5)    // FCクイックパテ（袋）
const tape  = Math.ceil(area / 60)      // ファイバーテープ（巻）

// 工程2: 下塗り
const under = Math.ceil(area / 3.3)     // ベストアンダー（袋）

// 工程3: 主材塗り
const main  = Math.ceil(area / 4.5)     // MURA WALL主材（袋）
const col1  = main                      // MURA WALLカラー（袋）= 主材と同数

// 工程4: フィニッシュ
const fin   = Math.ceil(area / 12.5)   // MURA WALLフィニッシュ（缶）
const col2  = fin                       // MURA WALLカラー（袋）= フィニッシュと同数

// 工程5: クリヤー（オプション・選択時のみ）
// 艶あり・艶消し共通ロジック
const cans16 = Math.floor(area / 80)
const remain = area - (cans16 * 80)
const cans2  = remain > 0 ? Math.ceil(remain / 10) : 0
```

### 6-2. 外装（MURA WALL EX）計算式

```javascript
const area = 施工面積（㎡）

// 工程1: シーラーW（コスパ最安自動選択・価格TBD時は数量のみ表示）
// ※価格が確定したらコスパ比較で最安組み合わせを自動算出
const sealer15 = Math.floor(area / 125)
const sealer_remain = area - (sealer15 * 125)
const sealer4  = sealer_remain > 0 ? Math.ceil(sealer_remain / 33.3) : 0

// 工程2: 主材
const mainEX = Math.ceil(area / 4.5)   // MURA WALL EX主材（袋）
const col1EX = mainEX                   // MURA WALL EXカラー（袋）

// 工程3: フィニッシュ
const finEX  = Math.ceil(area / 10)    // MURA WALL EXフィニッシュ（缶）※10㎡固定
const col2EX = finEX                   // MURA WALL EXカラー（袋）

// 工程4: クリヤー（オプション・内装と同ロジック）
```

## 9. 施工要領書の内包方針

アプリ内で「▼ 施工要領を確認」として折り畳み表示する。
内容はHTMLにテキストとして内包（PDF参照ではなく、重要箇所のみ抜粋）。

### 内装（MURA WALL）表示内容
1. 下地処理（パテ・テープ・ベストアンダー）の要点
2. 主材の混練り比（主材3.6kg + ボンド0.275kg + カラー0.1kg + 清水1.7L）
3. フィニッシュの混練り（フィニッシュ3.0kg + カラー0.1kg）
4. 各工程の乾燥時間目安

### 外装（MURA WALL EX）表示内容
1. シーラーW塗布の要点（原液使用・3時間以上乾燥）
2. 主材の混練り比（主材4.0kg + 専用ボンド1.2kg + カラー0.1kg + 清水1.2L）
3. フィニッシュの混練り（フィニッシュ3.0kg + カラー0.1kg）
4. 各工程の乾燥時間目安

### クリヤー表示内容
1. 2回塗り（3時間以上間隔）
2. 使用ローラー：マイクロファイバーローラー中毛
3. ⚠️ 内装に使用すると防火認定外
